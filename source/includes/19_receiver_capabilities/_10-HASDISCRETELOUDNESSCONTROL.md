### Signature

`<has_discrete_loudness_control></has_discrete_loudness_control>`


| Parameter | Description |
| --- | --- |
| bool | True/False |


### Example

```
<capabilities>
   <has_discrete_loudness_control>true</has_discrete_loudness_control>
</capabilities>
```
