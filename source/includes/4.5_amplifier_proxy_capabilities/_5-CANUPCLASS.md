## can \_  upclass

Specifies if the device switch can up convert. For example: Composite -\> S-Video 


### Signature

`<can_upclass></can_upclass>`


| Parameter | Description |
| --- | --- |
| bool | True/False |


### Example

```xml
<capabilities>
   <can_upclass>true</can_upclass>
</capabilities>
```
