## has toad input select

Capability of toggling through inputs


### Signature

`<has_toad_input_select></has_toad_input_select>`


| Parameter | Description |
| --- | --- |
| bool | True/False |


### Example

```xml
<capabilities>
   <has_toad_input_select>true</has_toad_input_select>
</capabilities>
```
