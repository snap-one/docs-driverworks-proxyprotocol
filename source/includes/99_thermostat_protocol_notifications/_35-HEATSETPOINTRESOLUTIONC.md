## HEAT\_SETPOINT\_RESOLUTION\_C

What increment to use (ie 1,3,5,10,etc), will do the same thing as the [`setpoint_heat_resolution`][1] capability.


### Signature

`HEAT_SETPOINT_RESOLUTION_C ()`


| Parameter | Description |
| --- | --- |
| num | Default 1 |


### Returns

`None`
